A wordpress plugin which is simply allow you to Display popup flash message after login.

== Description ==

Features:

*	Can be display popup flash message after user logged in.


== Installation ==

1. Upload `popup-message` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
